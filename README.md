# Neotec F&B Suite (neotec_fnb)

ERPNext/Frappe v15 compatible app (MVP):
- POS Order -> auto Sales Invoice
- Table Session + Multiple Checks + Shared Allocation
- KDS Ticket from POS
- Built-in docs pages at /neotec-docs

Install:
bench --site <site> install-app neotec_fnb
bench --site <site> migrate
